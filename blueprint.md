# Project Blueprint: Lets-Get-It (복불복 게임)

## Overview
"Lets-Get-It" is a framework-less web application for a "Fortune/Penalty" (복불복) game. It features a spinning roulette wheel where users can add players and penalties to decide who gets a mission.

## High-Quality AdSense Optimization (New)
The site has been optimized to meet Google AdSense's high-quality standards:
- **Comprehensive Content**: Added detailed "About" and "How to Use" sections to provide user value and context.
- **Clear Navigation**: Implemented a fixed navigation bar for easy access to all site sections and legal pages.
- **Essential Pages**: Created dedicated `about.html`, `contact.html`, and `privacy.html` pages to ensure transparency and trust.
- **Privacy Compliance**: Detailed Privacy Policy explicitly covering Google AdSense and cookie usage.
- **SEO Optimization**: Structured semantic HTML with proper headings and meta tags for better search engine visibility.

## Features & Design
- **Roulette Wheel**: A canvas-based interactive wheel that spins and selects a winner.
- **Participant Management**: Dynamic addition and removal of players.
- **Penalty/Mission Management**: Customizable penalties that are randomly assigned to the winner.
- **Visual Effects**: Confetti animation and modal results for an engaging experience.
- **Responsive Design**: Adapts to different screen sizes (Mobile/Desktop) with a modern UI.
- **Theme Support**: Includes both Dark Mode and Light Mode with persistence via `localStorage`.

## Technical Implementation
- **Architecture**: Multi-Page Application (MPA) for better SEO and AdSense compliance.
- **Security**: Strict Content Security Policy (CSP) and secure HTTP headers.
- **Styling**: Modern CSS variables, backdrop-filter, and flexible layouts.
- **Logic**: Vanilla JavaScript with event delegation and Canvas API.
